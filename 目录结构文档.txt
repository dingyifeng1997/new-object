目录结构:

index.html  :   未登录首页
login.php   :   登录页面
backLogin   :   登陆后首页
registrationProtocol    :   注册协议

public  :   公共文件夹
    bs  :   公共文件夹下的bootstrap包
    css :   公共文件夹下的首页/PC端/移动端样式表包
    img :   公共文件夹下的图片包
    js  :   公共文件夹下的登录前首页和登陆后首页逻辑代码

PC      :   PC端计算页面
    indoorColorXC		室内全彩型材
    indoorColorYZL		室内全彩压铸铝
    monochrome5.0		单色5.0
    monochromeP3.75		单色3.75
    monochromeP10		单色P10
    outdoorColorXC		室外全彩型材
    outdoorColorXT		室外全彩箱体
    outdoorColorYZL		室外全彩压铸铝
    pc404               未找到页面后显示

mobile  :   移动端计算页面
    indoorColorXC		室内全彩型材
    indoorColorYZL		室内全彩压铸铝
    monochrome5.0		单色5.0
    monochromeP3.75		单色3.75
    monochromeP10		单色P10
    outdoorColorXC		室外全彩型材
    outdoorColorXT		室外全彩箱体
    outdoorColorYZL		室外全彩压铸铝
    m404               未找到页面后显示

mysql   :   登录页面及数据库管理
    delUserMysql.php        删除数据库的数据
    getCaptcha.php          php生成验证码返回至前端(已弃用)
    matchOldPassword.php    用户修改密码,查找数据库.判断输入旧密码是否正确
    modifyUserMysql.php     数据库后台页面发送至此修改内容
    register.html           用户注册页面
    registry.php            注册页面链接数据库,插入注册数据
    selectTable.php         注册页面时,输入用户名查找账户是否存在
    setAccount.html         用户修改密码页面
    setAccountPhp.php       用户修改密码发送至mysql数据库
    showMysql.php           显示数据库信息
    skip.php                注册成功页面
    verify.php              登录页面判断数据库是否匹配

cookie  :   设置cookie值
    addCookie.php       添加一个登录成功Cookie
    checkCookie.php     返回当前cookie的值
    delCookie.php       删除cookie
