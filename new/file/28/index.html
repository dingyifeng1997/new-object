跨足VR技术 利亚德与川大智胜合作

2018-09-20 13:04:59

利亚德公告，3月22日，公司与四川川大智胜软件股份有限公司（以下简称“川大智胜”）签订了《“虚拟现实技术创新与应用”战略合作协议》。

　　双方合作内容如下：拟共同研究将LED小间距显示技术与VR技术融合并应用；拟共同投资建设和运营基于文化艺术与科学技术相结合为核心的平台（如校园电影院线），形成集产学研为一体的完整的高校影视动漫创新创业产业链；拟共同投资研发影院级“高清晰立体LED显示”相关技术；拟共同投资建设和运营“虚拟现实科普体验馆”。

　　川大智胜成立于1994年6月，注册资金150,417,397元，经营范围包括软件、硬件及配套系统开发，系统集成和图象图形工程，从事安防产品生产、销售及安防工程设计、施工；建筑智能化工程设计、施工；民航空管工程及机场弱电系统工程施工；电子工程、电子工程施工；电子及通信产品生产、销售；技术咨询、技能培训等。

　　利亚德表示，川大智胜在VR/AR技术和应用方面具有很强的影响力，利亚德在LED显示方面的技术和产品具有很强的优势，并且双方均将VR/AR应用作为发展战略的一个重要方向。双方的战略合作将会加快LED显示技术在VR/AR产业的创新和发展，对扩大利亚德LED小间距产品应用市场，有积极的作用。
