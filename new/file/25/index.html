苹果未来的VR头显或将采用微型LED显示屏

2018-09-20 13:03:00

在2014年5月苹果收购微型LED显示屏技术行业中的佼佼者LuxVue公司后，凯鹏华盈公司的风险投资者John Doer评论道LuxVue将实现显示屏技术上的突破。大家纷纷猜测苹果将怎样利用这项技术，但目前尚无头绪，尤其是苹果宣布2017的 iPhone手机屏幕供应链换成OLED后，Apple Watch的屏幕也改为采用OLED。所以在苹果帝国的版图中，微型LED显示屏位于何处？

　　在本周发行在EETimes杂志的一篇报告中，我们看到迄今为止对苹果动向最靠谱的推测。报告写道，微型LED技术将会用于“直投式显示屏”，包括应用在增强现实和虚拟现实领域，例如头显。EXALOS公司 CEO 表示对于下一代显示屏，微型LED将是提高效能和亮度的关键元件。

　　去年六月EETimes的另一篇报道指出高亮度和视觉增强系统如平视显示器和头戴式显示器可以提高航空和汽车等行业的安全系数和性能，因为飞行员和司机可以通过视线范围内的显示屏获得关键的导航数据和信息，而不再需要低头查看显示器。

　　对于消费者来说，增强现实智能眼镜或投影设备可以提供导航、提高安全系数、浏览广告和提供其他信息等作用。LED微型显示屏较为理想的使用方式是用于佩戴式系统，因为LED微型显示屏拥有所占空间小，耗能少，对比度高以及亮度超高的特点。

　　微型LED显示屏主要的应用由Apple projects负责。平视显示屏可能用于Project Titan，因为PrimeSense公司（被苹果收购）其他相关的技术可以助其一臂之力。另外，头戴式的显示屏将用于苹果秘密打造的虚拟现实头显。

　　最后，尽管没有确凿的证据，至少目前我们清楚微型LED技术的去处。这项先进的技术在不久的将来会出现在某种消费者版和专业版产品的身上。
