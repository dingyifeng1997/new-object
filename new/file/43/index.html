亿光推出红外NIR-C19M系列，适合虹膜辨识应用

2018-09-20 13:18:40

全球LED光电产业的龙头制造厂，亿光电子工业股份有限公司【TSE：2393】推出NIR-C19M系列为高效红外LEDs，采用效果对比最明显的波段810nm，辨识精准快速且散热性佳，特别适合安防设备(如门禁解锁)和手持装置的虹膜辨识应用。

随着市场需求与高科技的发展演进，现行许多系统与装置应用结合“身体就是密码”概念，以身体器官的独特性取代密码（生物辨识），如指纹、声音、虹膜、人脸等，辨识快速方便，且可免于传统记一长串密码的问题，其中非接触式光学辨识”虹膜辨识”传感器体积小、安全性高、辨识速度超快，可谓大家争相追逐的新一代技术。

亿光新品NIR-C19M系列为非接触式光学虹膜辨识，采用独特铸模制程，相较于传统灌胶制程的红外线LED产品，表面平整度提升许多，可靠度/信赖性大幅提高，使用特殊红外波段810nm波长，搭配ISP图像处理器和CMOS Sensor接收经人眼反射红外光，达到虹膜辨识功用，特殊光学设计可达到高反射率，小角度（FOV=25），在1000mA操作下辐射强度可高达2,700mW/Sr，热阻4.38℃/W，散热效果优异于市面上同属性应用之LED组件，在安全规范上，已通过国际IEC62471安全认证，保证对于人眼及皮肤不造成伤害。再加上体积仅3.5*3.5*2.3mm轻量化设计的优势，NIR-C19M系列，整体便利性更佳，特别合乎对机构设计空间要求较高的客户。

亿光结合专业研发、业务及市场团队，在加上超过34年以上的专业开发能力，在封装技术上不断追求创新与突破，以提供客户各种无可挑剔的产品方案。
