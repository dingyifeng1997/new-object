明纬推出多款新品，增添照明市场多元性

2018-09-20 13:17:56

产品一：HBG-240-DA系列 240W具DALI功能圆形天井灯电源供应器(具PFC)

产品简介：

因应天井灯市场对于DALI功能的应用需求，明纬新推出HBG-240-DA（240W）系列。此款系列除沿袭天井灯构造圆形机体易于安装外，并结合时下智慧调光市场需求，增加DALI灯控系统功能，符合国际标准IEC62386-101、102、207，大幅增加灯具系统设计的弹性。其他功能包含了短路保护、过电压保护、过电流保护、过温度保护、5年保固外，此系列通过了UL/CUL/ENEC/ CCC/CB/CE等国际认证，确保产品使用上安全无虞。合适的应用包含LED天井灯、工矿灯、舞台灯、投光灯与户外建筑灯光照明系统等。

产品特性：

恒流模式+恒压模式输出
明纬专利金属圆形外观，class I 设计（专利证号：CN201220314551）
内置主动式PFC功能
IP67/65防护等级，户内户外安装均可
功能可选：输出内部电位器调整,三合一调光,DALI
工作温度：-40~+85℃
尺寸(直径*高)—ψ191.5x69mm
5年保固

产品二：FDLC-80系列~ 80W恒功率输出LED驱动器

产品简介：

为满足经济型低成本泛光灯市场需求，明纬新推出FDLC-80系列。该电源采用恒功率设计，电源输入范围：180~295VAC.定电流宽范围输出：1000~2100mA，全机型灌胶，防水等级高达：IP67. 该系列产品符合国际LED相关安规认证EN55015. EN61547. EN61000-3-2,3. GB17743. GB17625.1等。为室内或户外照明应用提供绝佳解决方案。

产品特性：

输入范围：180~295VAC
超薄设计,高度：31.5mm
低空载功耗： <0.5W
PFC设计，PF≧0.9
工作温度： -30~+90℃
保护种类：、短路保护、过温保护
符合ENEC/CCC/CB/CE 认证
尺寸 (L x W x H)—151*53*31.5mm
3年保固

产品三：IDLV/IDPV-25系列~ 25W塑胶壳/PCB型恒压PWM输出LED驱动器（具PFC）

产品简介：

明纬已上市之多款室内专用LED驱动器并广受市场好评。塑胶外壳型IDLV家族及PCB板型IDPV家族，在设计规划上皆采用恒压PWM型输出模式 ，并具有主动式PFC功能。为使客户有更多小功率产品选择，明纬向下延伸推出一款25W的IDLV-25及IDPV-25系列。

为满足室内灯具与安装的需求，两系列采共板设计，仅外型与机构差异，电气规格皆相同。IDLV与IDPV系列产品均可接受90~295VAC全范围输入、并提供12V~60V恒压输出的多款机型选择、最高工作效率可达84%；符合UL8750、ENEC EN61347-1,-2-13、EN62384及GB19510.1,.14等LED照明类的国际安规认证。标准品即内建二合一调光功能(另提供具12V/50mA辅助电源输出的选购机型)，使得LED灯具系统业者在产品设计规划时具有更大的弹性设计空间。此两系列产品非常适用在LED平板灯、LED泛光灯等应用。

产品特性：

90~295VAC全范围输入
Class II设计
内建主动式功率因数矫正功能
内建二合一调光功能（0~10VDC和 PWM讯号）
低空载功耗：Blank-Type <0.5W，A-Type <1.2W
工作温度：IDLV-25:-20~+90℃(Tc), IDPV-25：-20~+50℃(Ta)
保护种类：短路保护、过电流保护
UL/ENEC/TUV/CCC/CB/CE 认证齐
全尺寸(长 x 宽 x高)—IDLV-25: 110 x 75 x 22mm/IDPV-25: 114.5 x 72.5 x 20 mm
3年保固
