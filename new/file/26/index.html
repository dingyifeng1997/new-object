台塑集团跨业携手”智” 造，打进专业照明世界杯

2018-09-20 13:03:38

台塑集团旗下南亚光电于2013年中正式从居家照明积极转型为工业及商业照明，2014年底推出Total Solution商业模式，2016年成功跨业结盟，抢下前瞻照明应用的滩头堡，于台湾国际照明展一举推出医疗专用智能平板灯、智能飞碟灯及公共安全的智慧监控路灯等，尤以智能飞碟灯(UFO Intelight)技压群雄，拿下2016年创新产品奖殊荣，成为本次展会吸晴亮点。

　　本次展会，台塑集团集结南亚光电、台塑商用车及长庚医院桃园失智症中心，将石化园区、医疗专区及商用车组装厂搬进展场了，透过最真实的实境show+Total Solution案例导览，让客户及观展者有更身历其境的感受，了解如何让照明质量提升的同时又能节电省钱。

　　LEDinside于2016 LED Taiwan展会中，访谈南亚光电罗荣晃副总经理，其表示：2016年营运策略则朝向开发全球经销伙伴，争取外销市场为主。未来南亚光电的服务也不再只是产品销售，将从Total Solution深化为System Solution，聚焦在灯光设计、施工管理与销售服务的全面结合，无论客户的新建、扩建或换装等照明需求，还可提供照明配管工程的加值服务。

　　展会亮点之一：智能平板灯、智能飞碟灯

　　南亚光电与长庚医院失智症中心共同开发【环境照光】医疗照明系统，主要用意在补足日光照射，促进失智者健康照护，而这智能平板灯最大特点就是可调光又可调色，有256种变色模式，色温从1700K-6200K，是医疗区域照明的新尖兵。

　　而荣获创新产品奖的智能飞碟灯(UFO Intelight)，极光舞影、唱作俱佳，有如声光魔术师，可唱歌、可调光、还可大玩256种情境变色功能，相当适合长照机构、养生村等医疗体系的使用。

　　※桃园长庚失智症中心徐主任莅临现场分享

　　本次展会亮点之二：台塑IVECO商用车厂实境展示

　　南亚光电将组装厂的概念搬到展场里，创下展会先例，以实境模拟呈现工业照明的点灯效果，此外，南亚光电从6m-30m高工业厂房、ShowRoom、车辆保修厂都有丰富的经验，搭配影片播放，吸引潜在客户目光。

　　本次展会亮点之三：智慧监控路灯

　　为打造安全的工作环境、提升公共空间照明，南亚光电本次也展示具监控功能的新型态路灯，结合监控与智能电控，未来将会在厂区或工业园区率先导入应用。



　　最后，罗荣晃副总提到：今年将提出大型新建或改建环境照明的系统方案，将服务范围扩大为“System Solution”，无论在危险区域、工业厂房或者医疗、商办空间等照明服务，将全面升级，请大家拭目以待。
