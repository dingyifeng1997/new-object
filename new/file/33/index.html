舞台LED显示屏设计的三种基本形式形式

2018-09-20 13:06:50

在不同的时期，戏剧家们都把当时最新的理念、最新的科学技术运用的舞台上来，展现在观众面前，自从电的出现加速了舞台技术和舞台设计的革新。舞台灯光从煤油灯时代转入了卤钨灯时代，再到后来出现的气体放电灯，再到电脑灯时代，舞台设计也从原来的传统的舞台转向机械式的舞台布景。近年来随着LED技术的出现和发展，舞台灯光也转向LED的时代，舞台设计也把LED很快的运用到了舞台设计和布景里，LED大屏幕出现在了世界上各种的舞台演出中。

　　现在一般出现在舞台上的大屏幕的设计，基本上分三种。

　　第一种：单单把大屏幕作为舞台设计的全部，大屏幕设计为背景、侧幕，场景中的空间时间一切都让大屏幕完成，演员做无实物的表演。

　　这样的情景很多地方都是这么来使用，这样做当然有它的好处，即省事又省钱，一张图片，把剧情里该交代的东西，都可以交代了。一段视频画面把可以把整个节目的形式感都出来了，特别是一些地方电视台，买了一个大屏幕，省了一大笔的舞美费用。

　　但是这样做，难免会造成整个舞台很概念化，今天要演农村场景，背景换张农房图片，明天演城市，换张高楼大厦图片，来段动感音乐，放段动感抽象的图案。毫无舞台的特性，也让舞台显的很单薄，毫无立体感，观众稍稍偏点角度，就看到一个个平面，说不定还带点反光。

　　这样的舞台形式做的最好的应该是中央台的春节联欢晚会了，整个舞台，上下左右都是大屏幕，像是做了个箱式布景，所以大的平面中的小块屏幕还可单独升降。当节目需要高山时舞台台面上的小屏幕可通过各自的升降先相处山的形状再在这样的基础上，加上立体的山的图片，背景布景呈现远山当做背景，顶上的大屏幕显示天空的形态，再加上视频制作团队对画面视频的精心处理，组成立个立体的逼真的、虚拟的理想空间，而且空间的图像可以是运动的。

　　从远景看，舞台既整体又大气，但是从近景看，LED大屏幕难免还是会出现颗粒感和不真实感，产生间离的效果，这样做还有一点的欠缺的是留给灯光设计的发挥空间太小，因为是箱式布景，给灯光安装的位置是极其有限的，所以舞台上会缺少气氛光、环境光，使舞台也就缺少了场景的氛围。这样的舞台造价高昂，所有单单用LED屏幕搭建好的舞台，也很难呈现完美的舞台效果。

　　第二种：根据舞台的框架以LED屏幕为主体，在各单位屏幕的周围留出灯光的位置的舞台。

　　这样的舞台也比较的常见，像东方卫视的“加油好男儿”的晚会，搭建了舞台整体纵深式的舞台框架，屏幕错落有致的悬挂在这样的舞台框架之上，在屏幕与屏幕之间挂上了摇头电脑灯。正是屏幕这样的纵深错落有致的排列，把舞台的消失的引向了远方，使整个画面的景深变长，产生很强的透视效果，加上边上留着这么多的给灯光悬挂和发挥的空间，整个舞台也不失好的气氛和动感，也表达出了这个晚会的主题。但是，这样做舞台结构会显得比较单一，不生动。

　　第三种：以LED大屏幕作为舞台的“中心”，或者是舞台的大背景，加上前面的实景，所组成的舞台布景。

　　2012年的中央电视台中秋晚会，设计更是巧妙。福州的实地古建筑的上空用LED屏幕搭建了一个大的月亮，晚会以建筑前的空地作为舞台，建筑做为背景，以圆月作为演出的主要元素，舞台设计构思巧妙，使LED屏幕跟实景浑然一体，符合晚会主题，而且舞台的形式感和立体感很强烈，也给了灯光有足够的空间，这样的一个舞台即符合传统的审美观念，也增加了现代科技元素，无论是从近景远景都是很真实，呈现出比较完美的舞台效果。

　　当然舞台的风格各式各样，每个时代都会有不同的舞台表现的形式，但LED无疑是已经成为一个时代的潮流，在生活的每个角落随处可见，在舞台更是不可缺少。应用科学技术不是唯一的手段，关键是我们能运用什么的手段来实现理想的舞台效果。
