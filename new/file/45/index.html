葳天科技推出新一代COB G-II系列，大幅提升亮度及CRI

2018-09-20 13:19:24

台湾高功率LED厂─葳天科技，推出性能大幅提升的COB G-II系列新产品，总计4个系列11个品种。

2015年，葳天率先业界推出2步MacAdam椭圆分档的COB，G-II系列沿续精准的2步MacAdam分档外，更藉由重新选定芯片等材质，将亮度大幅提升5-10%，瓦数分布自7W~260W，涵盖球泡灯~水银灯的应用范围，可依照所应用的灯具做更适当的搭配。


（图片来源：ProLight）

随着市场对于COB的规格要求越来越高，葳天研发处处长表示，COB G-II系列除了精准的色温分BIN外，此次更将显指提升到CRI 95以上 (自然白/暖白)，应用于摄影棚灯光时，可提供最佳的显色效果。在G-II外型设计上，沿用现有产品的外观尺寸以及电路，可简易地置换现有的COB系列，而不需要改变驱动电路以及LED周围元件等，是最佳的光通量升级方案。当然，全系列通过LM-80及 TM-21的可靠度认证。


（图片来源：ProLight）

葳天科技深耕LED照明产业超过12年，COB产品除了一般照明外，也推出商用照明的foodfresh / true white系列，提供更多不同照明的可能性。
