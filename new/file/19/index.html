除智能照明技术外，欧司朗未来还将关注这些领域......

2018-09-20 12:59:06

周二，欧司朗首席执行官Olaf Berlien在慕尼黑举办的“欧司朗光电半导体市场和技术日”上表示：“不管是在战略还是财务方面，欧司朗在未来的技术和趋势上都处于非常有利的地位。我们正在成为领先的光子学冠军”，“我们的光电半导体部门正逐渐成为我们战略发展的技术支柱。”

欧司朗一直在推动关注快速增长的高科技市场。除智能照明技术外，未来的发展将集中在可视化、传感和处理等市场领域，其中光对人和植物的生物效应将成为主要焦点。欧司朗正在朝着这个方向推出其产品组合，提升其作为高科技供应商的形象。目前，LED产品已经占合并销售额的70％左右。在此基础上，欧司朗希望发展成为光子学的冠军。

欧司朗光电半导体业务部门的产品组合可在自动驾驶、园艺和建筑控制等领域实现创新应用。从中长期来看，这些未来领域在整个行业周期中提供了巨大的增长机会。目前，由于货币浮动和汽车行业的不确定性，该部门的增长率暂时低于前几年的水平。

从2020年起，欧司朗光电半导体预计固定货币水平的收入增长率约为10％，调整后的EBITDA利润率在23％至29％之间。欧司朗首席执行官Olaf Berlien表示， “从长远来看，我们的前景仍然向好。尽管我们预计光电半导体近期会因市场影响而出现一些放缓，但是短期内对市场发展的可见度有限”。 对于2018/19财年，计划收入增长预计在5％到10％之间，调整后的EBITDA利润率在22％到26％之间。

新型光子学解决方案可以推动增长，例如具有更高分辨率的无眩光像素远光灯，以及节能的LED园艺灯，使植物能够在室内栽培并且可以提高产量和蔬菜质量。技术驱动因素包括LED的进一步小型化和创新的3D传感器解决方案，例如智能手机中的面部识别。
