隆达电子汽车应用市场再下一城

2018-09-20 13:18:58

台系隆达电子车前灯专用CORE系列产品除向中国大陆车厂tier1灯具供应商推广外，同时推出可适用于LED车灯光源产品的三晶、四晶、六晶封装元件。据悉此次深圳举办的AATIF秋季展上，多家大陆知名厂家展出使用隆达电子元件的产品。其中广州正澳电子一口气推出七款最为引人注目。

据悉，展会期间隆达电子予正澳电子将签署战略合作协议，未来十个月内，隆达电子将提供不少于2KK的CORE系列产品给予正澳电子；另外，隆达电子将在第三季推出一款专为LED车灯光源产品的元件，无论在发光效率上(lm/W)、单位面积发光亮度(lm/Area)上、耐热表现上都超过市售产品，未来将优先提供给正澳电子使用。

隆达电子车用CORE系列产品是采用散热系数最高的陶瓷基板，搭配倒裝芯片以及专利的荧光粉贴片技术，这种独特高效的线型光源正是LED车灯光源产品的最佳选择。

市场上LED车灯光源产品良莠不齐，隆达电子有和日本后装品牌合作的经验，去年底一口气在中国大陆市场上推出高、中、低阶产品，不但解決客户端亮度、耐热，光型等问题，并且提供打件、光学等设计支持，获得中国大陆知名品牌的认同与青睐一点都不意外。

隆达电子是台湾地区唯一拥有芯片、封装、模组、成品的上下游一贯的LED光源制造商。成立车用部门三年来，一直兢兢业业地耕耘，今年出击中国大陆车用市场，就一举斩获大订单。
