LED屏企六大上市企业18Q1业绩汇总

2018-09-20 12:59:51

业绩变动原因

01 艾比森

艾比森业绩的上涨，主要受益于公司显示屏订单的快速增长，本期公司实现营业收入3.34亿元，同比增长近70%，由此公司业绩实现快速增长。此外，在2018年第一季度，公司取得LED显示屏订单约4.23亿元，同比增长约29%；显示屏酒店运营业务实现营业收入约1,383万元，同比增长约108%。同时，虽然2018年第一季度公司业绩实现了快速增长，但是由于汇率波动导致本期出现汇兑损失约1,581万元，不可避免的影响了公司业绩的更高增长。

02 利亚德

利亚德业绩的变动，主要来源以下两个方面。首先，在智能显示在创意显示的带动下,营业收入较上年同期增长近40%，夜游经济及文旅新业态营业收入均倍增，VR体验实现50%以上增长。其次，小间距电视技术有了新突破， Supersafe产品已在量产中,预计将于二季度投放市场。MiniLED将在时机成熟时量产并投放市场。截至报告披露之日,小间距电视新签订单6.9亿元,其中境内渠道销售新签订单2亿元,“千店计划”已经实施70余家,计划年内实施300家。

03 洲明电子

洲明电子在2018年度第一季度，受人民币兑美元升值因素的影响，公司产生的汇兑损失约为2,200万元，再加上公司整体规模扩大，本季度管理费用增加约一倍，对公司归属于上市公司股东的净利润产生一定影响。但是在期内，公司积极拓展销售渠道，与去年同期相比，销售收入增长约60%，这也是公司净利润上涨的关键原因。

04 奥拓电子

2018年第一季度业绩变动的主要原因为公司营业收入规模扩大及订单增加所致，随着政府提出夜游经济建设,全国景观亮化需求爆发，公司收购双甲企业深度受益，一季度累计中标金额约3.98亿元,为公司全年业绩提供保障。

05 雷曼股份

报告期内，雷曼股份归属于上市公司股东的净利润较上年同期有所下降。主要原因是公司为促进LED 行业产品技术的升级换代，加大了公司自主创新知识产权产品COB小间距高清显示面板的研发和人员投入，导致本期成本与费用增加，其效益将于后期逐渐释放。同时，因汇率波动带来的汇兑损失及毛利率下降，对净利润产生了一定的影响。

06 联建光电

2018年第一季度联建光电净利润相较去年有所下降，主要来自于公司自 2017 年起打造集团管控管理平台，聚焦户外与线上媒体的精准投放，导致管理费用增长、技术研发费用有所增加；其次，受公司业务规模、融资规模不断扩大的影响，报告期内公司财务费用相比去年同期增长较大；最后，受户外广告经营环境影响，公司数字户外板块业务一季度营业收入有所下降。

2018年LED显示屏行业发展新趋势

01 新技术助力 小间距产品持续升级

从各大企业的业绩变动原因分析中可以看出，小间距在LED显示屏行业中发展势头依旧持续上涨，占据龙头地位，成为了拉动企业业绩上涨的主要板块。在未来，各大企业将会更加注重小间距产品的升级，优化，增加企业的产品竞争优势，特别是随着COB封装技术与MIRCO LED技术的不断成熟，为小间距LED显示屏带来更多的发展空间，进一步扩大了小间距在LED显示屏市场的占有率。接下来，各大LED显示屏企业将会继续加大在小间距领域上的布局，提高企业净利润增长比率。

02 应用场景多元化 为企业带来新契机

此外，在今年的报告中，我们也看到了新鲜的市场。在政府的高度推动下，全国夜游经济迅速爆发，带动景观照明市场的快速增长，据预测，2017中国景观亮化市场达680亿元,预计2018年可达780亿元，同比增长15%，庞大的百亿市场为重视显示和照明业务的企业带来了巨大的利润空间。与此同时，酒店、体育、金融等领域对LED显示屏的需求不断扩大，为LED显示屏提供了更加多元化的应用场景，也给竞争激烈的LED显示屏企带来更多机会，因此，LED显示屏企业应当注重LED显示屏的研发，以提供差异化产品提高竞争力

企业之间竞争越发激烈，各大龙头企业积极的投入到产品、市场、渠道、技术创新等领域的研发，虽说一方面扩大了龙头企业与中小型企业之间的差距，加剧了少数企业寡头垄断的格局，但与此同时，此举措也进一步促进LED行业产品技术的升级换代。
